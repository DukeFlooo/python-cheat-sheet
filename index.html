Python-Einstieg mit Thonny

Cheat Sheet für die 10. Klasse

Was machen wir heute?

In der ersten Stunde lernst du die Programmierumgebung Thonny kennen. Danach schreibst du dein erstes Python-Programm und speicherst Werte in Variablen.

Lernziele:
  • Thonny öffnen und wichtige Bereiche erkennen
  • Ein erstes Python-Programm mit print() schreiben
  • Verstehen, was eine Variable ist
  • Einfache Werte speichern und ausgeben

1. Die Thonny-IDE

Eine IDE ist eine Programmierumgebung. In Thonny schreibst du deinen Code, speicherst Dateien und führst Programme aus.

Aufbau von Thonny

Input und Output in Thonny

Wichtig:
  • Editor: Hier schreibst du den Python-Code.
  • Shell: Hier erscheinen Ausgaben und Fehlermeldungen.
  • Run: Damit startest du dein Programm.

2. Dein erstes Programm

Mit print() kann Python einen Text ausgeben.

print("Hello World!")

Die Ausgabe in der Shell lautet:

Hello World!
Mini-Aufgabe: Ändere den Text so, dass dein Name ausgegeben wird.

3. Variablen

Eine Variable ist wie eine beschriftete Box. In dieser Box kann ein Wert gespeichert werden.

name = "Lena" print(name)

Hier wird der Text "Lena" in der Variablen name gespeichert. Danach wird der Inhalt ausgegeben.

Noch ein Beispiel

alter = 15 print(alter)

Variablen können also unterschiedliche Werte speichern, zum Beispiel Texte oder Zahlen.

4. Mehrere Variablen zusammen benutzen

name = "Tom" alter = 16 print("Ich heiße", name) print("Ich bin", alter, "Jahre alt.")

Python kann Texte und Variablen zusammen ausgeben.

Mini-Aufgabe: Ersetze Tom und 16 durch deine eigenen Daten.

5. Merkhilfe

Begriff Bedeutung Beispiel
print() Gibt etwas in der Shell aus print("Hallo")
Variable Speichert einen Wert name = "Mia"
Text / String Steht in Anführungszeichen "Hallo"
Zahl Wird ohne Anführungszeichen geschrieben 15

6. Kleine Übungsaufgaben

  1. Gib mit print() den Text "Python ist interessant" aus.
  2. Speichere deinen Namen in einer Variablen und gib ihn aus.
  3. Speichere dein Alter in einer Variablen und gib es aus.
  4. Erstelle zwei Variablen: fach und note. Gib beide aus.

7. Das solltest du nach der Stunde können

  • Thonny bedienen
  • Ein Python-Programm starten
  • print() benutzen
  • Einfache Variablen anlegen und ausgeben
