Thonny

Programmierumgebung

In Thonny schreibt man Python-Code, speichert Programme und startet sie.

  • Editor: Hier schreibt man den Code.
  • Shell: Hier sieht man die Ausgabe und Fehlermeldungen.
  • Run: Damit wird das Programm ausgeführt.

Wichtig am Anfang

Merkhilfen
  • Python achtet genau auf die Schreibweise.
  • Klammern und Anführungszeichen müssen stimmen.
  • Fehlt ein Zeichen, gibt es oft eine Fehlermeldung.
  • Programme darf man ausprobieren und schrittweise verändern.

Ausgabe mit print()

Erster Python-Befehl

Mit print() gibt Python etwas in der Shell aus. Das kann zum Beispiel ein Wort, ein Satz oder später auch der Inhalt einer Variable sein.

print("Hello World!")

Wie liest man das?

  • print ist der Name des Befehls.
  • Die Klammern gehören dazu und müssen mitgeschrieben werden.
  • Der Text steht in Anführungszeichen, weil es ein Text ist.
Man kann an dieser Stelle schon sagen: print() ist eine Funktion. Für den Anfang reicht aber die einfache Erklärung: Es ist ein Python-Befehl, mit dem man etwas ausgeben kann.

Beispiele

print("Hallo") print("Ich lerne Python") print("Mein Name ist Mia")

Variablen

Werte speichern

Eine Variable ist ein Name für einen gespeicherten Wert. Man kann sich eine Variable wie eine beschriftete Box vorstellen.

name = "Lena" print(name)

Hier wird der Text "Lena" in der Variablen name gespeichert. Danach wird der Inhalt ausgegeben.

Noch ein Beispiel

alter = 15 print(alter)
In Python muss man Variablen am Anfang nicht extra deklarieren. Man weist ihnen einfach einen Wert zu.

Datentypen am Anfang

Einfach angerissen

Variablen können unterschiedliche Arten von Werten speichern. Für den Einstieg reichen diese drei Arten.

Art des Werts Woran erkennt man ihn? Beispiel Wichtig zu beachten
String
Text		 steht in Anführungszeichen "Hallo" Ohne Anführungszeichen ist es kein Text.
Integer
ganze Zahl		 ohne Anführungszeichen 15 Keine Nachkommastellen.
Float
Kommazahl		 mit Punkt statt Komma 3.5 In Python schreibt man 3.5 und nicht 3,5.

Beispiele für Variablen mit verschiedenen Datentypen

name = "Tom" alter = 16 note = 2.5 print(name) print(alter) print(note)

Typische Anfängerfehler

Darauf sollte man achten
  • Text ohne Anführungszeichen: print(Hallo) ist falsch.
  • Fehlende Klammer: print("Hallo" ist falsch.
  • Kommazahl mit Komma: 3,5 ist nicht dasselbe wie 3.5.
  • Groß- und Kleinschreibung: Print() ist nicht dasselbe wie print().
